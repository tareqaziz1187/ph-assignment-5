Question-2: What is the difference between innerHTML, innerText, and textContent ?

Difference between innerHTML, innerText, and textContent: 1.innerHTML Returns or sets the HTML content inside an element, 2.innerText Returns or sets only visible text inside an element, 3.textContent Returns or sets all text content, including hidden elements.

Question-3: What is event delegation in the DOM?

Event delegation is a technique where a parent element handles events for its child elements using event bubbling. Instead of adding event listeners to multiple child elements, you add a single listener to their parent.

Question-4: What is event bubbling in the DOM?

Event bubbling is the process where an event triggered on a child element propagates up to its parent and ancestors in the DOM hierarchy. How It Works? An event occurs on a child element., The event bubbles up through its parent elements to the root ().
